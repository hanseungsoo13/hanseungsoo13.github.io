Vision Language Model

트래블로그: 사진 기반 여행 블로그 생성기

Naver Boostcamp · KCI 논문 2025.01 - 2025.02

Overview

Vision Language Model(VLM)과 Large Language Model(LLM)을 결합하여 사진만으로 여행 블로그를 자동으로 생성하는 시스템입니다. 이 연구는 한국컴퓨터정보학회 논문지(KCI)에 투고되었습니다.

Key Contributions

  • VLM(Qwen-VL)과 LLM(하이퍼클로바) API를 활용해 LLM의 시각적 한계 극복 및 문장 생성 능력 극대화
  • 프롬프트 엔지니어링과 파인튜닝만으로 기존 LLM 대비 10% 이상 향상된 양질의 블로그 생성
  • AI Safety 확보를 위한 안전장치 구현
  • VLM 활용 캡션 생성 전처리 수행
  • 멀티 프로세싱을 통한 추론 시간 단축 및 효율적인 GPU 활용 기여

Technical Approach

본 프로젝트에서는 두 단계의 파이프라인을 구축했습니다. 첫 번째 단계에서는 Qwen-VL 모델을 사용하여 입력 이미지들로부터 상세한 캡션과 맥락 정보를 추출합니다. 두 번째 단계에서는 추출된 정보를 하이퍼클로바 LLM에 전달하여 자연스러운 여행 블로그 글을 생성합니다.

특히 프롬프트 엔지니어링을 통해 블로그 글의 스타일, 톤, 구조를 제어하고, 파인튜닝을 통해 도메인 특화된 성능을 확보했습니다.

Results

제안한 시스템은 기존 LLM 기반 방법 대비 블로그 품질 평가에서 10% 이상의 향상을 보였으며, 사용자 평가에서도 높은 만족도를 기록했습니다.
